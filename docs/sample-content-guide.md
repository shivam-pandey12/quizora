# Quizora Sample Content Guide

Phase 2 intentionally avoids an automatic seed script so no service account secrets or accidental production writes are introduced.

Use `/admin/categories`, `/admin/quizzes`, and `/admin/quizzes/{quizId}/questions` to enter sample content manually.

## Categories

- General Mastery: Balanced rounds for fast recall, pattern spotting, and broad confidence.
- Science: Concept-first quizzes across physics, chemistry, biology, and space.
- Technology: Programming, systems thinking, internet culture, and product logic.
- History: Timelines, civilizations, policy shifts, and decisive world moments.

## Quiz Ideas

- World Logic Sprint: A sharp general knowledge set built for speed, recall, and confident elimination.
- Science Signal: Physics, biology, and chemistry prompts with clean pacing and clever traps.
- Code Composure: Programming fundamentals, web logic, and careful debugging instincts for builders.

## Question Examples

- Single choice: Which planet has the strongest surface gravity among the terrestrial planets?
- Multiple choice: Which statements describe a pure function?
- True/false: A published private quiz should appear on public quiz listing pages.
- Text placeholder: Name the concept represented by a stable URL-safe quiz identifier.

Publish only after adding at least one active question.
