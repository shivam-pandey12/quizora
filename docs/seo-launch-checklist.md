# SEO Launch Checklist

## Files

- Open `/robots.txt` and confirm it points to the production sitemap.
- Open `/sitemap.xml` and confirm public routes, quizzes, and categories are present.
- Confirm private routes are absent from the sitemap.
- Confirm protected routes use noindex metadata: dashboard, profile, play, results, billing private pages, admin, creator, classroom, assignments, quick match/status, and private room flows.

## Public Indexable Pages

- `/`
- `/quizzes`
- `/quizzes/[slug]`
- `/categories`
- `/categories/[slug]`
- `/leaderboard`
- `/rooms`
- `/pricing`
- `/privacy`
- `/terms`
- `/refund`
- `/contact`

## Search Console

- Add the production domain property.
- Verify ownership.
- Submit `/sitemap.xml`.
- Inspect homepage, one quiz page, and one category page.
- Check robots and mobile usability.

Indexing is not instant. Search engines decide crawl timing.
