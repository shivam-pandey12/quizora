# Plan Gating

Phase 13 gates premium features gently. Free users can still browse quizzes, play basic public quizzes, view leaderboards, join basic flows, and use account/support pages.

Premium gates may appear around:

- Advanced progress analytics.
- Larger or more frequent room creation.
- Quick-match expansion.
- Creator quiz tooling.
- Classroom scale.
- Assignment exports.
- Premium profile/dashboard surfaces.

The UI must show upgrade cards or usage meters instead of crashing. Firestore rules protect billing records, but full server-side feature enforcement for all product actions is future hardening.
