# Refund Policy Notes

Quizora Phase 13 includes a practical `/refund` placeholder. It must be reviewed before live payments.

Current behavior:

- Admins can record refund requests manually.
- Razorpay `refund.processed` webhooks can mark payment/refund summaries.
- Automatic refund API calls are not implemented in Phase 13.

The policy should explain that paid plans are time-limited digital access passes and do not include cash prizes, wallets, betting, or payouts.
